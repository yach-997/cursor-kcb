# 一次性配置（只需做一次）：
# 1. 打开 https://my.shipstatic.com → 账户/设置 → 创建 API Key
# 2. 打开 https://github.com/yach-997/susuc-kcb/settings/secrets/actions
# 3. 新建 Secret：名字填 SHIP_API_KEY，值粘贴刚才的 Key
# 4. 告诉我「配好了」，或自己在 Actions 里点一次 Deploy
# 之后每次我推送更新，同学刷新原链接即可，不用换地址、不用通知。
